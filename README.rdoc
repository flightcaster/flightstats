= Description

  FlightStats is a gem that provides a wraper over the FlightStats Web Services
  API, specifed at https://www.flightstats.com/developers/bin/view/Web+Services/web_services_directory

= Usage

	require 'flightstats'
	
	FLIGHTSTATS_GUID 	= 'my_guid'
